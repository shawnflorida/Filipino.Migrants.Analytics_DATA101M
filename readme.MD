## âš¡ Getting Started

### Prerequisites
- Python 3.7+
- pip (Python package manager)

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/shawnflorida/Filipino.Migrants.Analytics_DATA101M
   cd Filipino.Migrants.Analytics_DATA101M
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Streamlit app**
   ```bash
   streamlit run Home.py
   ```

4. **Open your browser** at `http://localhost:8501` to explore the dashboard.


5. Open the website online at https://dataviz101m---final-project-k2vml2puzghqjcvnhgbmae.streamlit.app/SupplementaryDataset
